# EV Battery Pack Performance Analysis

## Overview
Designed and analyzed a 16S1P LiFePO4 battery pack with BMS integration. Performed charge-discharge testing and developed data-driven insights.

## Key Highlights
- SOC (State of Charge) estimation using current integration
- Efficiency analysis (~76%)
- Safety monitoring (overvoltage/undervoltage)
- Interactive dashboard using Power BI

## Tools Used
- Python (Pandas, Matplotlib)
- Power BI
- Excel
